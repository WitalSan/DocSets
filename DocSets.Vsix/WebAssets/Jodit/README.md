# Эксперимент Jodit

В каталоге закреплена локальная UMD-сборка Jodit `4.13.8` для
экспериментального HTML-редактора DocSets `[EJ]`.

Исходные файлы:

- `https://unpkg.com/jodit@4.13.8/es2021/jodit.min.js`
- `https://unpkg.com/jodit@4.13.8/es2021/jodit.min.css`

Jodit распространяется по лицензии MIT:

- `https://github.com/xdan/jodit/blob/main/LICENSE.txt`

Редактор работает полностью локально внутри WebView2. Обращение к CDN во время
работы расширения не требуется.
