# Эксперимент CKEditor

В каталоге закреплена локальная UMD-сборка CKEditor 5 версии `48.3.1` для
экспериментального HTML-редактора DocSets.

Исходные файлы извлечены из официальной self-hosted ZIP-сборки:

- `https://cdn.ckeditor.com/ckeditor5/48.3.1/zip/ckeditor5-48.3.1.zip`

Облачную CDN-сборку нельзя упаковывать внутрь VSIX: она помечена каналом `cloud` и
при использовании локального GPL-режима переводит редактор в read-only.

Контрольные суммы SHA-256:

- `ckeditor5.umd.js`: `D592C9FE65852BEC702B24A8D3839D52D6C4A756760E837A487594DA6DBD3BE8`
- `ckeditor5.css`: `E61CBE94FBA222B803218B201DEA7378461C0F6E64A8F6DA45C276B22501EA8A`

Сейчас редактор запускается с `licenseKey: 'GPL'` только для оценки технологии.
Перед распространением DocSets необходимо отдельно подтвердить совместимость
лицензии всего продукта с GPL 2+ либо приобрести коммерческую лицензию CKEditor.
Платные плагины в экспериментальную сборку не подключены.
