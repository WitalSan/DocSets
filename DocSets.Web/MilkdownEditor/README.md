# Экспериментальный редактор Milkdown

Этот каталог содержит исходный код отдельного редактора заметок Milkdown/Crepe.
Рабочий TOAST-редактор не зависит от этой сборки и продолжает открываться кнопкой `E`.
Milkdown открывается кнопкой `EM`.

## Сборка

Для воспроизводимой сборки используются зафиксированные версии из `package-lock.json`:

```powershell
npm ci
npm run build
```

Команда формирует локальные файлы `milkdown-editor.js` и
`milkdown-editor.css` в `DocSets.Vsix/WebAssets/Milkdown`. Они встраиваются
в сборку расширения как ресурсы, поэтому редактор не обращается к CDN и
работает без сети.

При обновлении пакетов необходимо пересобрать bundle, проверить изменения
лицензионных комментариев в конце JavaScript-файла и прогнать
`MilkdownClipboardIntegrationTests`.

## Граница хранения

В документе DocSets по-прежнему хранится канонический Markdown. Служебное
представление Milkdown создается только внутри редактора. В частности,
масштаб блочного изображения хранится в fragment `docsets-scale`, а исходный
`alt` временно кодируется в `docsets-alt`, потому что ImageBlock использует
Markdown-alt для собственного коэффициента размера.
