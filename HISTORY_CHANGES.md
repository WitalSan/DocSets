# History — плоский журнал переходов

- Запись создаётся сразу при обнаружении перехода в другой символ или файл.
- Ограничение просмотра в 10 секунд удалено.
- История остаётся плоским списком обычных symlink-элементов.
- Повторная цель не дублируется: существующая запись обновляется и переносится в начало.
- Переход, выполненный из History, не добавляет новую запись.
- Для символов создаётся Symbol-ссылка, для файлов без разрешённого символа — File-ссылка.
- Для перемещений внутри одного символа новая запись не создаётся.
- Для участков вне символов переход учитывается на уровне файла, поэтому обычная прокрутка файла не засоряет журнал.
- Максимальный размер истории увеличен до 2000 записей.
- Опрос активной позиции выполняется каждые 500 мс.
